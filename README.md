Remindly – Smart Reminder App

Remindly is a simple and modern Reminder Application built with React Native.
The app allows users to create, edit, and manage reminders while receiving notifications at the scheduled time.

📱 App Screens

The application includes the following main screens:
	•	Login Screen – User authentication
	•	Home Screen – Displays all reminders
	•	Add / Edit Reminder Screen – Create or modify reminders
	•	Profile Screen – User profile information
	•	Settings Screen – App preferences including language, notifications, and privacy policy

  🛠 Technologies Used
	•	React Native
	•	JavaScript (ES6)
	•	React Navigation
	•	Node JS
